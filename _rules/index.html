Vuewin Game Mechanics for 2N, 4D and 6/49:

  1. When the 2N Game is selected, user will be able to select any two (2)-digit number combination from 01 to 31 (01,01 to 31,31).  User can either do this manually or with the random button which will automatically generate two (2) random numbers. The “jackpot” or the winning combination follows the exact order on the number chosen in the equivalent PCSO draw.

  1. When the 4D Game is selected, user  will be able to select four (4) digits from 0 to 9, duplicates are allowed. User can either do this manually or with the random button which will automatically generate four (4) digits. The jackpot combination follows the exact order on the number chosen in the equivalent PCSO draw in which the number count of the matches will start from the rightmost digit.

  1. When the 6/49 Game is selected, user will be able to select six (6) numbers from 1 to 49, with no duplicates among the 6 numbers. User can either do this manually or with the random button which will automatically generate the six (6) numbers.
